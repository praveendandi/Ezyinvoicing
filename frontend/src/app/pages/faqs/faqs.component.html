  • SAC/HSN Codes 1. What is a sac code ?

    Steps

    A classification system developed and issued by the (CBIC) Service Tax Department of India to classify various services is known as SAC or Services Accounting Code. These codes are used to classify services for better measurement, taxation, and recognition of services. These codes are controlled by the (CBEC) central board of excise and customs.

         2. How to add the SAC/HSN codes in Ezy Invoicing ?

    Steps

    • 1. Go to SAC/HSN codes list
    • 2. Click on Add
    • 3. Fill the following information required to create a SAC/HSN code in the system
    S.no Field Definition
    1 Description Enter Opera transaction description
    2 Transaction code Enter Transaction code of description
    3 Taxable
    Select Yes if the description is taxable (Eg. Room Charges)
    Select No if the descriptions in not taxable (Eg. Liqour)
    4 Type Select SAC or HSN
    5 Sac/HSN code Enter the SAC/HSN code of the description
    6 Net
    Select Yes if the taxes are included in the price
    Select No if the price in excluding of taxes
    7 SEZ exempted
    Select this options exempts the taxes for the SAC code
    De selecting this applies the taxes to the SAC code
    8 Service charge Select Yes if the description has Service charger or else select No
    9 Service charge net Select Yes if the service charge applies on Net price or else select No
    10 Service charges taxes applies
    This option will be visible only if you select service charge yes.
    Select Apply from parent –Service charges of the parent item will be applied.
    Select Separate GST if the description has separate Service charge > Enter SC sac code > Enter SC GST Tax Rate (%)
    Select No tax – if taxes are not applicable for the service charge
    11 CGST Enter CGST value
    12 SGST SGST value autofill once the CGST value is given
    13 IGST Enter IGST Value(* if the description has CGST and SGST then IGST field gets disabled and vice versa)
    14 State Cess rate Enter State Cess rate if any
    15 Central Cess rate Enter central cess rate if any
    16 VAT Rate Enter VAT rate if any
    • 4. After enter all the required details please click on save
    • You can use Edit option provided under actions column if you want to make any changes to particular SAC/HSN code.

         4. How to handle error invoice?

    Steps

    • 1. Go to invoice list
    • 2. Find the error invoice and click view
    • 3. The error is displayed on the screen
    • 4. You can use either Re upload or Reprocess option.

    Re- Upload

    • 1. Click on reupload
    • 2. Upload new invoice
    • 3. If the newly uploaded is in the right format than invoice status changes to pending and you can now generate IRN for that invoice

    Re-process

    • 1. Click on Re – process
    • 2. The system will re process the uploaded invoice.
    • 3. If there is no error found after re processing than the status of the invoice will be changed pending and you can now generate IRN for that invoice.
  • Error Handling 1. How to solve SAC Code not found error ?

    If you find this error in an invoice that means a particular SAC code or payment type is not available in your SAC code or payment type master data.

    To solve this issue

    • 1. Click on add button provided beside the error
    • 2. If you add it under payment type click on payment type and fill the required details
    • 3. If you add it under SAC/HSN Codes click on sac and fill the required details. Click here to check guidelines to add a SAC/HSN codes.
    • 4. After adding the SAC/HSN code,Click on Re-Proccess .Now the invoice status would change to pending
    • 5. Click on Generate IRN.
         2. How to solve Invalid GST number error ?

    If you find this error in an invoice that means the GST number in provided in the invoice is invalid .

    To solve this issue,

    1. Click on edit in tax payer details
    2. Enter valid GST number and save
    3. Click on Re-Process and generate IRN
